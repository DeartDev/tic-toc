# Tic Tac Toe (3 en Raya)

Juego clásico de 3 en Raya implementado en un solo archivo HTML con JavaScript y CSS.

## Características

- Interfaz oscura (dark theme)
- Tablero interactivo de 3x3
- Sistema de puntuación (Victorias X, Victorias O, Empates)
- Personalización de nombres de jugadores
- Detección automática de ganador
- Reinicio del tablero sin perder puntuación

## Uso

Simplemente abre el archivo `tic_toc.html` en cualquier navegador web.

```bash
# Ejemplo en Termux
termux-open tic_toc.html
```

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)

## Desarrollo

Este proyecto fue desarrollado utilizando:
